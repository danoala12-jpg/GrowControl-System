# GrowControl-System

> A modern Python application for monitoring and controlling automated environments.

## 📖 About

GrowControl-System is a Python-based application designed to monitor sensors and control devices through a modern graphical interface.

The project is built as a learning project during my retraining as an IT Specialist for System Integration.

## 🚀 Planned Features

- Dashboard
- Sensor Monitoring
- Temperature & Humidity
- Device Control
- Automation
- Data Logging
- Settings
- Modern UI

## 🛠 Technologies

- Python
- CustomTkinter
- SQLite
- Git
- GitHub
